# MagnetarPrometheus Scripts

This directory contains operational scripts for local development and validation.

Expected baseline scripts:

- `run_backend.sh`: start the backend entrypoint
- `run_tests.sh`: execute the test suite with coverage enforcement
- `bootstrap_python.sh`: prepare the Python runtime environment

The exact implementations can evolve, but the repository should always provide a simple script-driven way to run the product and its validation paths.
